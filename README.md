# 口袋妖怪 - 像素冒险 (Pokemon H5 Game)

## 项目概述

使用纯原生 HTML5 + Canvas + JavaScript 开发的像素风俯视角 2D 宝可梦 RPG 游戏，无需任何框架，直接在浏览器中运行。

## 技术栈

- **语言**: 原生 HTML5 / CSS3 / ES6+ JavaScript（零依赖）
- **渲染**: HTML5 Canvas 2D API
- **地图**: 程序生成 Tilemap（内置真新镇 + 1号道路）
- **数据**: PokéAPI（第一世代151只，含 localStorage 缓存）
- **存档**: localStorage（3槽存档 + 自动存档）
- **平台**: PC（WASD/方向键）+ 移动端（虚拟摇杆）

## 快速开始

直接用浏览器打开 `index.html` 即可（支持本地服务器）：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`

## 操作说明

### PC 键盘
| 操作 | 按键 |
|------|------|
| 移动 | WASD / 方向键 |
| 确认 | Z / Enter |
| 取消 | X / Escape |

### 移动端（自动识别触屏）
- 左下角虚拟摇杆：移动
- A 按钮（红色）：确认
- B 按钮（蓝色）：取消

## 游戏特色

- 像素风 GBA 色调，Canvas 分辨率 480×320
- 程序生成地图（真新镇 + 1号道路），含碰撞层与触发器
- 草丛随机遭遇系统
- 完整回合制战斗（速度先手、伤害公式、STAB、暴击）
- 18种属性克制关系
- 6种异常状态（中毒/剧毒/烧伤/麻痹/睡眠/冰冻）
- 精灵球捕捉系统
- PokéAPI 数据集成 + localStorage 缓存
- 等级升级 + 进化系统
- 3槽存档 + 自动存档
- PC + 移动端双平台支持
